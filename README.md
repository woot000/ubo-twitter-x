# woot000's Twitter/X uBlock Origin Filters

A set of filters for [uBlock Origin](https://github.com/gorhill/uBlock) which makes Twitter/X mildly more usable by reducing some visual clutter, removing unnecessary buttons and hiding superfluous advertisements.

## Install

To install the filters, click on [`twitter-x.txt`](twitter-x.txt) and save it to your desktop. Then, navigate to `uBlock Origin` > `Open the dashboard` > `My filters`, click on `Import and append...`, and open `twitter-x.txt`. The filters should appear in the filter box, appended at the bottom if any other filters were there before.
